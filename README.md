## Comment démarrer le projet
> [!WARNING]
> Vous devez dans un premier temps avoir npm, go et Node installé sur votre machine
> <br></br>Veuillez également suivre les instructions de configuration dans le ReadMe du backend 
#### Frontend
```
cd frontend
npm install
npm run dev
```
#### Backend
```
cd backend
go mod tidy
go run main.go
```
